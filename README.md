# Получаем праздничне дни
Программа получает спсиок праздничных дней за указанный год и страну с помощью внешнего API.

## Переменные окружения
Для работы скрипта нужен API-ключ от сервиса ч праздничными днями (например, Calendarific и др.)
1. Создайте файл .env в корне проекта
2. Добавьте в .env ваш API-ключ в следущем формате:
```
env
API_KEY = ваш_api_ключ
```

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Как запустить проект
После установки, проект можно запустить комадной:
```
python main.py
```
