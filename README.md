# Получаем праздничне дни
Программа получает спсиок праздничных дней за указанный год и страну с помощью внешнего API.

## Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Как запустить проект
После установки, проект можно запустить комадной:
```
python main.py
```

## Цель проекта
- Научиться работать с HTTP-запросами
- Освоить библиотку request
- Разработано в рамказ школы "Третье место"
